Pipeline

How It Works

A clean two-phase pipeline: first you define parking spaces interactively, then the system watches live video and counts free spots every frame using image processing.

01 / DEFINE
🖱️
Space Picker

Run Parking_Space_Picker.py and left-click to draw rectangles over each space on a static image. Right-click to remove. Saved via Pickle.

→
02 / PREPROCESS
🔲
Grayscale + Blur

Each video frame is converted to grayscale, then Gaussian blur smooths noise so pixel intensities are consistent for thresholding.

→
03 / THRESHOLD
⚡
Adaptive Threshold

Adaptive thresholding creates a binary image — white pixels represent edges/cars, black pixels represent empty asphalt. Robust to lighting changes.

→
04 / COUNT
🔢
Pixel Count Decision

Each space region is cropped and non-zero pixels counted. Below threshold → FREE. Above → OCCUPIED. Updated every frame.

🖱️
Interactive Space Picker

Click to annotate any parking lot image. No ML labelling tools needed — pure OpenCV mouse callbacks with live visual feedback.

🔲
Adaptive Thresholding

Uses cv2.ADAPTIVE_THRESH_GAUSSIAN_C to handle shadows and variable lighting without needing to re-calibrate for each environment.

⚡
Real-Time Frame Analysis

Processes every frame of the video feed, updating colour-coded overlays and the free/total counter live — no artificial delays.

💾
Persistent Positions

Space coordinates saved with Python's pickle module — annotate once, run the detector as many times as needed without re-marking.

🎨
Colour-Coded Overlay

Green rectangles for free spaces, red for occupied. Pixel count displayed inside each box via cvzone for instant interpretability.

🔄
Seamless Video Looping

When the video ends, the frame counter resets automatically — perfect for demos and presentations that need continuous playback.

Clean, Readable Code

Two focused Python scripts under 60 lines each — stripped of complexity so the core computer vision logic is front and centre.

main.py
Parking_Space_Picker.py
# main.py — Video processing + real-time space detection import cv2 import pickle import cvzone import numpy as np # Load saved parking space positions with open('car_park_pos', 'rb') as f: posList = pickle.load(f) width, height = 107, 48 # rectangle dimensions per space def checkParkingSpace(imgPro): spaceCounter = 0 for pos in posList: x, y = pos # Crop the thresholded image to each space region imgCrop = imgPro[y : y + height, x : x + width] count = cv2.countNonZero(imgCrop) # non-zero = white = car pixels if count < 900: # below threshold → FREE color = (0, 255, 0) # green thickness = 5 spaceCounter += 1 else: # above threshold → OCCUPIED color = (0, 0, 255) # red thickness = 2 cv2.rectangle(img, pos, (x + width, y + height), color, thickness) cvzone.putTextRect(img, str(count), (x, y + height - 3), scale=1, thickness=2, offset=0, colorR=color) # Overlay total free / total count cvzone.putTextRect(img, f'Free: {spaceCounter}/{len(posList)}', (100, 50), scale=3, thickness=5, offset=20, colorR=(0, 200, 0)) cap = cv2.VideoCapture('carPark.mp4') while True: if cap.get(cv2.CAP_PROP_POS_FRAMES) == cap.get(cv2.CAP_PROP_FRAME_COUNT): cap.set(cv2.CAP_PROP_POS_FRAMES, 0) # loop back to start success, img = cap.read() # Pre-processing pipeline imgGray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY) imgBlur = cv2.GaussianBlur(imgGray, (3, 3), 1) imgThres = cv2.adaptiveThreshold(imgBlur, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C, cv2.THRESH_BINARY_INV, 25, 16) imgThres = cv2.medianBlur(imgThres, 5) kernel = np.ones((3, 3), np.uint8) imgDilate = cv2.dilate(imgThres, kernel, iterations=1) checkParkingSpace(imgDilate) cv2.imshow("Image", img) cv2.waitKey(10)
# Parking_Space_Picker.py — Interactive space annotation tool import cv2 import pickle try: with open('car_park_pos', 'rb') as f: posList = pickle.load(f) # load existing positions if file exists except: posList = [] # start fresh if no saved data width, height = 107, 48 def mouseClick(events, x, y, flags, params): if events == cv2.EVENT_LBUTTONDOWN: posList.append((x, y)) # left-click: add space at cursor if events == cv2.EVENT_RBUTTONDOWN: for i, pos in enumerate(posList): x1, y1 = pos if x1 < x < x1 + width and y1 < y < y1 + height: posList.pop(i) # right-click: remove clicked space with open('car_park_pos', 'wb') as f: pickle.dump(posList, f) # auto-save after every click while True: img = cv2.imread('carParkImg.png') # reload static image each frame for pos in posList: cv2.rectangle(img, pos, (pos[0] + width, pos[1] + height), (255, 0, 255), 2) # magenta rectangle per saved space cv2.imshow("Image", img) cv2.setMouseCallback("Image", mouseClick) cv2.waitKey(1)
