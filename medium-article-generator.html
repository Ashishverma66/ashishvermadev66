What it does

An end-to-end pipeline: give it a topic, get back a fully researched, Medium-formatted article — automatically.

🔬
Academic Research
Pulls peer-reviewed papers directly from ArXiv, downloads PDFs, and surfaces the most relevant scientific findings for any topic.
🌐
Multi-source Web Crawl
Searches the web via DuckDuckGo, reads full article bodies with Newspaper4k, and scrapes HackerNews tech discussions.
📺
Video Intelligence
Reads YouTube transcripts and extracts timestamped insights from video content — turning hours of video into useful research.
🤖
Modular Agent Team
Each data source is handled by a specialist agent running GPT-4o-mini, coordinated by a GPT-4 team leader that synthesizes everything.
📝
Medium-Optimised Output
Articles are formatted specifically for the Medium platform — proper structure, subheadings, and engaging prose saved as Markdown.
⚡
FastAPI Web Interface
A clean web UI powered by AgentOS — just enter a topic, hit generate, and the entire research-to-article pipeline runs in the background.

The Agent Team

Nine specialized agents, each an expert in their data domain, orchestrated by a GPT-4 team leader.

🔬
arxiv-research-agent
Queries ArXiv for academic papers, downloads PDFs to the research papers directory, and summarizes key findings.
ArxivTools
🌐
web-search-agent
Performs general web searches, collects article summaries and resource links from across the internet.
DuckDuckGoTools
🔥
hackernews-research-agent
Accesses HackerNews to surface trending tech discussions, community opinions, and curated article links.
HackerNewsTools
📰
news-article-research-agent
Reads full article bodies from any URL, extracts body text and auto-generates summaries via Newspaper4k.
Newspaper4kTools
📖
wikipedia-research-agent
Provides encyclopedic background information, definitions, and cross-references for any topic.
WikipediaTools
🐦
x-research-agent
Searches posts on X (Twitter), includes engagement metrics, and surfaces real-time social sentiment.
XTools
📺
youtube-research-agent
Fetches video transcripts, metadata, and timestamps from YouTube to extract insights from video content.
YouTubeTools
✉️
gmail-agent
Drafts, reads, and sends emails through Gmail — useful for sharing generated articles directly (requires OAuth setup).
GmailTools
👑
medium-article-creation-team (Orchestrator)
GPT-4 team leader that delegates research tasks, synthesizes all findings, and produces the final Medium article draft.
GPT-4

How it works

From a single topic prompt to a fully written article — five stages, fully automated.

01
Topic Input
User enters a topic via web UI or Python
02
Task Delegation
GPT-4 orchestrator assigns research tasks to specialist agents
03
Parallel Research
Agents query ArXiv, HackerNews, YouTube, web & more simultaneously
04
Synthesis
Team leader compiles findings into a coherent draft article
05
Save & Confirm
Draft shown to user; saved as .md after confirmation

Core Architecture

A peek at how the agent team is assembled in code — clean, composable, and extensible.

app.py
Python 3.13
# Define orchestrator and agent models
orchestrator_model = OpenAIChat(id="gpt-4")
model = OpenAIChat(id="gpt-4.1-mini")

# Specialist agent — one of nine
arxiv_research_agent = Agent(
    id="arxiv-research-agent",
    name="Archive Research Agent",
    model=model,
    role="Arxiv Research Assistant",
    instructions=[
        "Search for research papers, authors and topics",
        "Summarize your findings clearly and concisely"
    ],
    tools=[ArxivTools(download_dir=dir_path)],
)

# Assemble the team with GPT-4 as team leader
medium_article_team = Team(
    name="Medium Article Creation Team",
    members=[
        arxiv_research_agent, web_search_agent,
        hackernews_research_agent, wikipedia_research_agent,
        youtube_research_agent, x_research_agent,
        news_article_research_agent, gmail_agent
    ],
    model=orchestrator_model,
    tools=[LocalFileSystemTools(target_directory=target_dir)],
    stream=True, markdown=True
)

# Wrap everything in AgentOS → FastAPI app
agent_os = AgentOS(
    name="Medium Article Generator OS",
    teams=[medium_article_team]
)
app = agent_os.get_app()

Quick Setup

Get the project running locally in under five minutes.

1
Clone the repository
Grab the source code from GitHub.
git clone https://github.com/Ashishverma66/Medium-Article-Generator-With-Agno.git cd Medium-Article-Generator-With-Agno
2
Install dependencies
Uses uv for fast, reproducible installs. Python 3.13+ required.
pip install -e .
3
Configure environment variables
Create a .env file in the project root with your API keys.
OPENAI_API_KEY=your_openai_api_key_here # Optional — for Gmail agent GMAIL_CLIENT_ID=your_gmail_client_id GMAIL_CLIENT_SECRET=your_gmail_client_secret GMAIL_REFRESH_TOKEN=your_gmail_refresh_token
4
Start the server
Launch the FastAPI web app and navigate to http://localhost:8000
python main.py
5
Or use it directly in Python
Integrate the agent into any Python script.
from app import main_agent response = main_agent.run("The Future of Agentic AI Systems") print(response.content)
